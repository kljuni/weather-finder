## Getting started
- create a .env.local file in root directory
- add your api key into the created env file in format ```VITE_WEATHERAPI=paste_your_api_key_here```

## Project setup

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```
